GiftFinder API

Buraya hem POST hem de GET isteği yapabilirsin, test amaçlı olarak ikisini de koydum kolay olsun diye. İki türlü de aynı sonuç dönüyor.
API URL'i : /api/1.0/
Hediye bulucu URL'i : /api/find_gift/

Gönderebileceğin Veriler

interests
hediye alınacak kişinin ilgi alanları
zorunlu mu : hayır
format : liste
GET örneği : /?interests=Internet&interests=Yemek
categories
hediyenin ait olduğu kategori
zorunlu mu : hayır
format : liste
GET örneği : /?categories=EvEsyasi
recipients
hediye alınacak kişi
zorunlu mu : hayır
format : liste
GET örneği : /?recipients=arkadas
age
hediye alınacak kişinin yaş aralığı
zorunlu mu : hayır
format : yas_alt_sınırı-yas_ust_sınırı
GET örneği : /?age=20-25
sex
hediye alınacak kişinin cinsiyeti
zorunlu mu : hayır
format : string
değerler : male, female, both
default değer : both, yani her ikisine de uygun hediyeleri döndürüyor bir değer göndermezsen
GET örneği : /?sex=male

Örnek GET İstekleri

http://dunyayidegistir.com/api/find_gift/?recipients=arkadas&interests=Futbol&interests=Internet&interests=Yemek&categories=Yiyecek&age=10-20
futbol, internet, yemek ile ilgilnen 10-20 yaş arası arkadaşlara alınabilecek hediyeler
